문서 보는법

자바 스크립트의 레퍼런스는 크게 2가지 

자바 스크립트 언어자체에서 제공하는 명령어들의 API

자바스크립트가 동작하는 호스트 환경의 API로 구분된다.
호스트는 웹브라우저, node.js 등등이 있다.

자바스크립트 API 문서
o	ECMAScript (표준문서)
o	자바스크립트 사전 (생활코딩)
o	자바스크립트 레퍼런스 (MDN)
o	jscript 레퍼런스 (MSDN)

호스트 환경의 API 문서
o	웹브라우저 API
o	Node.js API
o	Google Apps Script API
