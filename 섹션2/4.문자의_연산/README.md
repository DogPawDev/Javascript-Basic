문자의 연산

줄바꿈
\n은 줄바꿈을 할 수 있도록 도와준다.
```
alert("안녕하세요.\n생활코딩의 세계에 오신 것을 환영합니다");
```
문자와 문자를 더할 때
```
alert("안녕하세요.”+”생활코딩의 세계에 오신 것을 환영합니다");
```
문자열의 길이를 알고 싶을 때
“문자열 입력한 것들”.length 를 붙여주면 된다.

문자열에서 특정 문자를 찾고 싶을 때
```
“code”.indexof(“c”); // 결과값 0 
```
문자의 위치를 반환해준다. 컴퓨터는 0부터 시작이기 때문에 맨 처음 글자는 0 이다.
