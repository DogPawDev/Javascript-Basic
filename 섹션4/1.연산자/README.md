연산자

연산자는 컴퓨터에게 지시하기 위한 기호인데 이미 = 라는 대입 연산자를 사용해봤다.
```
a = 1 
```
다음과 같이 1을 a에 대입을 한다는 연산이다.

여기서 a는 변수이며 1은 상수이다. a는 좌항 1은 우항인데 여기서 좌항은 무조건 상수가 올 수 없다.



대입 연산자 말고도 비교 연산자라는 것이 있다.
```
alert(1==2) //false
alert(1==1) //true
alert("one"=="two") //false 
alert("one"=="one") //true
```
