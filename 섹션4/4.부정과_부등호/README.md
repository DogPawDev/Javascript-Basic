부정과 부등호
!=
```
alert(1!=2);            //true
alert(1!=1);            //false
alert("one"!="two");    //true
alert("one"!="one");    //false
```
>
<도 마찬가지임
```
alert(10>20);   //false
alert(10>1);    //true
alert(10>10);   //false
```
>=
```
alert(10>=20);      //false
alert(10>=1);       //true
alert(10>=10);      //true
```
! 부정의 이미  == 의 반대다 
1==1 은 참이지만 1 != 1 할 경우 부정이 된다.

!== 는 ===의 반대 되는 것이다.
말 그대로 ===의 반대이기 때문에 데이터 값과 타입이 같으면 반대의 참과 거짓을 반환 한다.
